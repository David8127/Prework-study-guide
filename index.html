HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Within the head element, you can add metadata to help the browser execute the code.
  • Metadata is not observable to the person viewing the final page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can change the color for an element using HTML and Hex color codes.
  • using an external CSS file is ideal for projects and keeping the other types of code separate.

Git

  • git allows programmers to edit code on a branch separate from the main branch.
  • git status: checks what branch we are currently on
  • git checkout switches you to a different branch
  • After pulling and merging branches via git, always make sure you update the local main branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript files are always executed from top to bottom
  • You can create an array to organize items that are of a similar topic
  • Functions are very useful in JavaScript code, but make sure to call a function in the code to execute it.
